## 第一步，放弃 gitbook

gitbook 更推荐使用他自己出的服务，而我的需求是本地部署

gitbook 本地部署的库很久没有维护了，举步维艰，所以替换为 honkit

## honkit

honkit 几乎完全兼容 gitbook 的能力，至少插件是都可以使用的

安装
```js
npm install honkit --save-dev
```
